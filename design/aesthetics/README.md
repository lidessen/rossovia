# Aesthetic Practice

**Status:** pilot
**Authority:** [decision 017](../decisions/017-aesthetic-practice-pilot.md)

This is a small practice area for making the project's productive, open, and
non-exploitative purpose perceptible in documents, interfaces, illustration,
and other public expressions. It is not a brand book, asset warehouse, visual
canon, or substitute for human judgment.

The portable [`visual-design`](../../skills/visual-design/SKILL.md)
skill carries the adaptive `cultivate`, `shape`, `design`, and `review` judgment established
by [decision 029](../decisions/029-visual-design-skill.md). This
directory remains the project's source-bearing local practice. Its seeds,
directions, tokens, and treatments never become portable skill defaults.

## Inheritance ladder

| Artifact | What it retains | Authority boundary |
|---|---|---|
| `seed/` | explicit Principal preferences and disconfirming contrasts | preference signal, not a shared direction |
| `cases/` | a task's audience, relation, sources, brief, and acceptance | proposed direction for one task |
| `studies/` | observed artifact-in-use, review, failure, and revision | evidence, not a reusable rule |
| `directions/` | a source-bearing project direction with explicit provisional or accepted status | future cases may inherit it; only repeated, reviewed evidence supports acceptance |
| `references/` | attributed external/source material and license/use limits | evidence for a case, never style authority |

Create a subdirectory only when its first real artifact exists. A provisional
direction may precede its first production case only when it retains a
comparative source field, named human preference evidence, and an observation
that would revise it. Do not stockpile images or copied styles. Every retained
reference must name what decision it changed, its source/license, and what must
not be copied.

## Practice loop

1. Before a continuing artifact family begins, form a small attributed source
   field, compare real alternatives with the Principal, and retain one
   provisional direction. Skip this step when grounded direction already
   exists or the work is a one-off local adjustment.
2. Start each real case from its audience, production object, intended action,
   and inherited direction.
3. Prepare an Aesthetic Case: relation, contrast, few references, material and
   accessibility constraints, negative boundary, and review question.
4. Lower it to a Work Cell Expression Brief only with actual tools/capabilities.
5. Evaluate the expression in use, retain a study, and route failure to the
   next smallest revision.
6. After representative cases, retain, revise, reject, or promote the
   provisional direction; only repeated, human-reviewed evidence changes its
   status to accepted.

Project-specific direction comes first. Use mature external design systems and
editorial guides as attributed fallback references, adapting their structures to
this project's users and mandate rather than copying their aesthetic surface.
The portable skill's
[built-in reference index](../../skills/visual-design/references/design-sources/index.md)
provides a small official-source fallback when a new host has no local corpus;
it does not replace the source-bearing records retained here.

The first retained sources are the [initial triptych](references/2026-07-10-initial-triptych.md)
and [Common Workshop masthead](references/2026-07-10-common-workshop-masthead.md).
They record Principal preference and accepted seed material, not a public visual
direction.
