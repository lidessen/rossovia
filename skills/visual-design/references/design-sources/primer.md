# GitHub Primer

## Use for

Complex web-product information architecture, focused responsive layouts,
readable type hierarchy, and semantic color tokens across themes.

## Decision lenses

- [Layout](https://primer.style/product/getting-started/foundations/layout/)
  frames layout around reading order, mental models, focus, responsiveness, and
  reduced friction.
- [Typography](https://primer.style/product/getting-started/foundations/typography/)
  demonstrates restrained type roles, readable line length, and hierarchy that
  does not depend primarily on color.
- [Color usage](https://primer.style/product/getting-started/foundations/color-usage/)
  separates primitive colors, functional roles, and limited component/pattern
  tokens across multiple themes.

## Transfer boundary

Recover semantic token layering, focus, and responsive information structure.
Do not inherit GitHub's token names, state meanings, centered regions, density,
palette, or component surface without equivalent domain consumers.
